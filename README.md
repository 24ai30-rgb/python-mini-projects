# 🚀 Python Mini Projects

A collection of beginner-friendly Python projects to practice core concepts like file handling, APIs, and automation.

---

## 📂 Projects

### 📄 PDF Merger
Merge multiple PDF files into a single file using PyPDF2.

### 🔳 QR Code Generator
Generate QR codes from any URL using Python.

### ⏰ Water Reminder
Simple script to remind you to drink water at intervals.

### 📰 News App
Fetch latest news using an API based on user input.

### 🎮 Quiz Game
CLI-based quiz game with multiple-choice questions and scoring.

### 🧮 Calculator
Basic calculator for arithmetic operations.

---

- Python

- requests
